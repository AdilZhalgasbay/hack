# figma-implement-design

Translate Figma nodes into production-ready code with 1:1 visual fidelity using the Figma MCP workflow (design context, screenshots, assets, and project-convention translation).

**Requires a working Figma MCP server connection.**

## Skills

| Skill | Description |
|-------|-------------|
| `figma-implement-design` | Structured workflow for translating Figma designs into pixel-perfect production code |

## Usage

```
/figma-implement-design:figma-implement-design
```

Provide a Figma URL and the skill will extract design context, capture screenshots, download assets, and translate everything into code following your project's conventions.

## Prerequisites

- A configured Figma MCP server connection
