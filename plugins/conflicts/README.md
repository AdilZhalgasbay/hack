# conflicts

Git merge conflict resolution agent. Analyzes the intent behind changes on both branches and makes informed decisions about what to preserve, requesting user input only when both sides have meaningful conflicting changes.

## Skills

| Skill | Description |
|-------|-------------|
| `conflicts` | Resolve git merge conflicts by analyzing change intent across branches |

## Usage

```
/conflicts:conflicts
```

Invoke when you have merge conflicts to resolve, need to sync a branch, or want to merge branches with intelligent conflict resolution.
